WHY DESIGNED THIS WAY?

1) Choice of colors:

- as expected it should be both contrasting and matched to brand colors. That is why the application combines distinct yellow, orange and black.

2) Rounding / no rounding shapes:

to available easierto recognize clickable elements on the page have been rounded. Information elements(non-clickable) are in rectangles withoutrounding.

3) Color groups:

to simplify available use the application, items have been grouped based on theirlocation in color group:
- navigation color
- text and description colors
- colorregarding tickets, payments, places

4) The startscreen due to the 2 basic valuesthat the machine should have, namely:

- attracting attention
- simplicity
It includes both an advertisement of events played, popularin a given period, as well asthe firststep in the process of buying a ticket / subscription - that is, choosing a ticket.

5) Instructions on the startscreen:

due to some recipients who are not familiar with the technology before starting the purchase process, they are instructed about the need fortouch interaction with the screen.

6) Top navigation:

to facilitate free navigation through the application, the top navigation has been designed in the interface, which allows you to easily return to even the main page of the application. In addition to it (if the user did notrecognize itsfunctionality) there is a bottom navigation that allows you to go back to previousscreens, and an icon in the upper right cornerthat allows you to return to the main page.

7) No scrolling on the page:

due to default low responsiveness of the screen for which the application was designed.

8) Descriptive elements written in black text on a white background:

with more text it isimportant that it is accessible to the eyes,so the text on the screen would imitate a piece of paper.

9) Default values:

due to the short interaction with the screen by the user, the value part is entered / selected by default and can be changed by the user.

10)Seat selection = ticket selection, 10 tickets = group ticket:

by reducing the required place and time (number ofsteps) to make a purchase,selecting a seat in the room, you reserve (by default) a normal ticket, and then with one click you can change the normal ticket to a reduced one. A group ticket is attached when choosing the 10th place automatically (on the page with the choice of placesthere isinformation about both tickets)
