Online Library Landing Page with signing up.

About the prototype

Landing Page with signing up form was created as exercise and to extend my portfolio. Shown interface is only Landing Page - it doesn't contain any subpages and ability to sign in. Prototype is responsive - I made Web App and desktop versio (1366 x 768).
During work I focused on graphic design and putting elements in the right place for users. In prototype I included error messages and some interactions to show how my proposition for intuitive solution on sign up for users.

Why design like this?

Arragement of social media icons:
Polish language users read pages from left to right - that's why I put the icons on left side, so that they won't miss it.

Chatbot placement:
Social media taught users that the chat is in right bottom corner - If the user was looking for chat - he would find it where he was looking for it.
Clickable elements:
I devided them into 2 groups:
1) Main Navigation - white, if mouse over - blue
2) Singing up, signing in, Sending something - różowe przyciski
Groups were created to make it easier to use website for users.

Elements stacking order on webpage:
Over the sign up button I showed a value that is given by the Library. (with ability to swipe immediatly to sign up form) and 3 more values to confirm the user that this service is the right choice.

Typeface
Library is focused on IT Business - that's why I chose "Fira Coda" typeface - which is font for programmers.
