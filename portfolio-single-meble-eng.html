Online Furniture Shop

ABOUT THE PROJECT

Here I present my own web and app design for E-commerce furniture shop. It's not a commercial project I've made it during this course. It was also a try for Adobe XD, which I didn't use before. The project contains 2 prototypes.
I've made mock ups of landing page, room page, shop page and item page for both web and iOS phone.
For web I've used the most common size of the screen 1366x768 pixels and as I'm the user of iPhone SE to see the design that I've made I used size 640x1136 pixels.
The grid that I've used is 8px and the margin is 20px.
